Computadores são máquinas incríveis.

Como disse Alan Turing, se é que disse mesmo:
As máquinas me surpreendem muito frequentemente.
Por exemplo, às 22:35 estava eu programando o seguinte código: 
        
        // A utility function to swap two elements
                
            void swap(int* a, int* b)
            {
            int t = *a;
            *a = *b;
            *b = t;
            }
Quando de repente um Australiano entra pela porta e me lança um:
The book is on the table?
Meio perdido, retribuí elevando ao cubo!
